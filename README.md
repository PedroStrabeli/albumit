AlbumIt

Aplicativo que visa pesquisar hashtags no instagram, listá-las e permitir ao usuário logado que adicione a um álbum pessoal, podendo então gerir seu álbum como desejar.



Clone o repositório.

Instale as dependências com "npm install"

Rode o aplicativo com "node bin/www" ou "nodemon bin/www".

O Aplicativo estará funcionando na porta 5000, como avisado no console ao iniciar a aplicação.

Para o funcionamento correto do aplicativo, deve-se ter o MongoDB instalado.
